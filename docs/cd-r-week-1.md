# Release Candidate

## Code/design review week 1 

Deze week zijn jullie je begonnen aan de laatste sprint voor de opdrachtgever: de release candidate. Je hebt deze week meer geleerd over het live zetten van een website en over het animeren met GSAP. 

### Aanpak

Voorafgaand aan deze Code/Design review heb je bepaald waar je graag feedback op wilt ontvangen. Je hebt dit verwerkt in een issue of een PR. Vandaag ga je feedback krijgen van minimaal twee andere groepen. Zie onderstaande tabel voor de indeling:

| AdConnect | Bajeslab | FootGuard | SnappThis |

| Miledoni | Nebula | Oncollaboration |
